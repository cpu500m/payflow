# Podongdong's pay 용어사전

> Podongdong's pay에서 사용되는 도메인 용어 정의.
> 엔티티 상세는 `도메인모델.md` 참조.

---

## 1. 결제 (Payment)

| **한국어** | **영어**              | **설명**                                   |
|---------|---------------------|------------------------------------------|
| 지갑      | Wallet              | 계좌와는 다른 개념. 사이버 지갑이라고 생각하면 됨.            |
| 가맹점     | Merchant            | 플랫폼과 계약을 맺고 결제를 수신하는 사업자.                |
| 충전      | Charge              | 사용자가 은행 계좌에서 지갑으로 돈을 옮기는 행위              |
| 결제      | Payment             | 사용자가 지갑에서 돈을 쓰는 행위 ( 가맹점에서 사용)           |
| 결제승인    | PaymentApproval     | 지갑 잔액을 확인하여 해당 결제를 확정짓는 행위               |
| 결제취소    | PaymentCancellation | 차감했던 금액을 정산 전에 다시 사용자의 지갑으로 돌리는 행위       |
| 환불      | Refund              | 차감했던 금액을 정산 후에 다시 사용자의 지갑으로 돌리는 행위       |
| 정산      | Settlement          | 플랫폼에서 가맹점 계좌로 수수료를 떼고 돈을 입금해주는 행위        |
| 알림      | Notification        | 결제 완료 등의 이벤트 발생 시 플랫폼이 사용자/가맹점에게 알려주는 행위 |
| 거래원장    | TransactionLedger        | 지갑에서 들어오고 나가는 모든 히스토리를 기록하는 장부           |
